# HBC Bootable HackMii Installer

A version of the **HackMii Installer** that can be launched directly from the **Homebrew Channel**.

## About

This project allows the HackMii Installer to be easily loaded from the Homebrew Channel without needing to use **LetterBomb** again.

This can be useful in situations such as:
- Accidentally deleting your **BootMii files** from your SD card
- Needing to **reinstall BootMii or the Homebrew Channel**
- Wanting a quick way to relaunch the installer without repeating the exploit process

This repository simply provides a version that can be **booted from the Homebrew Channel** for convenience.

## Usage

1. Download the files from this repository.
2. Place the folder inside the `/apps/` directory on your SD card.
3. Insert the SD card into your Wii.
4. Launch the **Homebrew Channel**.
5. Select **HackMii Installer** from the list of applications.

## Disclaimer

Use at your own risk. This repository only repackages the installer to make it easier to launch from the Homebrew Channel.
