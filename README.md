# HBC-bootable-HackMii-Installer
A version of the HackMii Installer bootable from the Homebrew Channel
THIS PROGRAM DOES NOT BELONG TO ME.
I simply wanted to make a verson of it that can be easilly loaded from the Homebrew Channel, so you don't have to use LetterBomb more than once.
For example, this can be useful if you accidentally remove the BootMii files from your SD card.
